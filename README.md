# 大学专业推荐平台

一个基于十五五计划和中国未来发展趋势的高中毕业生大学和专业推荐平台。

## 功能特性

### 🎯 专业评测
- **兴趣测评** - 了解兴趣爱好，发现潜在专业方向
- **人格测评** - 基于MBTI等科学模型分析性格特质
- **能力测评** - 评估核心能力，匹配适合的专业领域
- **价值观测评** - 探索价值观，找到匹配的专业

### 📝 智能问卷
- **梦想城市** - 选择理想的学习和工作城市
- **地区偏好** - 选择偏好的地区（华北、华东、华南等）
- **兴趣爱好** - 多选兴趣爱好（科技、艺术、商业等）
- **擅长领域** - 评估核心技能（数学逻辑、创新创造等）
- **家庭背景** - 了解经济状况，匹配适合的学校
- **性格特质** - 分析性格类型（内向/外向、分析/创造等）
- **工作场所偏好** - 选择理想工作环境
- **职业目标** - 设定职业期望（高薪资、稳定、成长等）

### ✨ 精准推荐
- **智能推荐算法** - 基于测评结果和问卷的个性化专业推荐
- **匹配度评分** - 科学的匹配度评分系统（0-100分）
- **匹配原因** - 详细说明推荐理由
- **数据可视化** - 薪资对比图表、雷达图等
- **专业详情** - 工作内容、职业路径、就业城市等详细信息
- **大学推荐** - 包含排名、地区、专业匹配度、学费等信息

### 📊 未来规划
- 职业发展路径规划
- 分阶段目标设定
- 未来趋势分析
- 基于十五五计划的发展建议

### 💰 行情与薪资
- 各专业薪资水平对比
- 职业发展路径可视化
- 地区薪资对比
- 薪资增长率分析

### 🤖 AI智能咨询
- **OpenAI集成** - 集成GPT-4o-mini模型，提供智能对话
- **上下文理解** - 结合用户问卷答案和测评结果
- **深度定制解答** - 个性化专业推荐和职业规划建议
- **实时对话交互** - 流畅的对话体验
- **专业问题解答** - 回答关于专业、大学、就业等各种问题

## 快速开始

### 安装依赖

```bash
npm install
# 或
yarn install
# 或
pnpm install
```

### 运行开发服务器

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本

```bash
npm run build
npm start
```

## 环境变量

创建 `.env.local` 文件并添加以下变量：

```env
# OpenAI API配置（用于AI智能咨询功能）
# 获取API Key: https://platform.openai.com/api-keys
OPENAI_API_KEY=your_openai_api_key_here
```

**注意**：如果不配置OpenAI API Key，AI聊天功能将使用备用回复系统，功能会受限。建议配置API Key以获得完整的AI咨询体验。

## 功能说明

### 专业评测系统
通过科学的测评体系，帮助用户了解自己的：
- 兴趣爱好倾向
- 性格特质类型
- 核心能力水平
- 价值观取向

### 智能推荐算法
基于以下因素生成推荐：
- 测评结果
- 问卷回答
- 专业匹配度
- 未来发展趋势
- 就业前景

### AI咨询功能
集成OpenAI GPT-4o-mini模型，提供：
- **专业问题解答** - 回答关于专业、大学、就业的各种问题
- **个性化建议** - 结合用户问卷答案和测评结果
- **职业规划指导** - 提供详细的职业发展路径建议
- **深度定制分析** - 基于用户背景的深度分析

### 推荐算法
智能推荐系统基于以下因素：
- **兴趣爱好匹配** - 根据用户兴趣爱好推荐相关专业
- **技能优势匹配** - 根据擅长领域匹配专业要求
- **性格特质匹配** - 根据性格类型推荐适合的专业
- **职业目标匹配** - 根据职业期望推荐薪资和发展前景匹配的专业
- **地区偏好匹配** - 根据梦想城市和地区偏好推荐大学
- **家庭背景匹配** - 考虑经济状况推荐合适的学校

### 数据可视化
- **薪资对比图表** - 柱状图展示不同专业的薪资水平
- **雷达图** - 展示专业的综合评估（匹配度、薪资、前景等）
- **职业发展路径** - 折线图展示薪资增长趋势
- **地区薪资对比** - 对比不同城市的薪资水平

## 使用流程

1. **专业评测** - 完成兴趣、人格、能力、价值观测评
2. **智能问卷** - 填写详细的问卷（梦想城市、兴趣爱好、技能等）
3. **查看推荐** - 系统基于测评和问卷生成个性化推荐
4. **查看详情** - 点击专业卡片查看详细信息（工作内容、职业路径等）
5. **AI咨询** - 与AI助手对话，获得更多建议和解答
